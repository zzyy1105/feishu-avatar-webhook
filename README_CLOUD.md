# 飞书群头像自动更新系统

当飞书多维表格中"发机通道申请状态"字段变更为"已通过"时，自动更新对应飞书群的头像。

## 🚀 部署方案

### 方案1: 本地运行（需要电脑开机）
适合测试和个人使用
- [查看本地部署文档](README.md)

### 方案2: 云端部署（推荐，电脑关机也能运行）
适合生产环境，7x24小时运行

#### 选项A: 免费云平台（零成本）
- **Render.com** - [部署指南](Render部署指南.md)
- **Railway.app** - 类似Render
- **Fly.io** - 免费额度

#### 选项B: 云服务器
- 阿里云/腾讯云/华为云
- 成本: 10-30元/月
- [查看部署脚本](deploy.sh)

#### 选项C: Webhook方案（最佳）
- 实时响应，零轮询
- [查看Webhook指南](Webhook部署指南.md)

## 📋 功能特性

- ✅ 自动监听多维表格变化
- ✅ 实时更新飞书群头像
- ✅ 完整的日志记录
- ✅ 支持多种部署方式
- ✅ 异常自动重试

## 🛠️ 技术栈

- Python 3.9+
- Flask (Webhook服务)
- 飞书开放平台API
- Docker (可选)
