# 简易上传方案 - 使用GitHub Desktop

## 为什么推荐GitHub Desktop？

- ✅ 图形界面，无需命令行
- ✅ 自动处理Git配置
- ✅ 拖拽即可上传
- ✅ 比命令行更简单

---

## 方案1：使用GitHub Desktop（最简单）

### 第1步：下载安装GitHub Desktop

1. 访问：https://desktop.github.com/
2. 点击 "Download for Windows"
3. 下载完成后双击安装
4. 安装完成后打开GitHub Desktop

### 第2步：登录GitHub账号

1. 打开GitHub Desktop
2. 点击 "Sign in to GitHub.com"
3. 输入你的GitHub用户名和密码
4. 授权GitHub Desktop

### 第3步：添加本地仓库

1. 点击 "File" → "Add local repository"
2. 点击 "Choose..." 按钮
3. 选择文件夹：`E:\AI\修改头像`
4. 如果提示 "This directory does not appear to be a Git repository"
   - 点击 "create a repository"
   - Repository name: `feishu-avatar-webhook`
   - 点击 "Create repository"

### 第4步：发布到GitHub

1. 在GitHub Desktop中，你会看到所有文件
2. 在左下角填写：
   - Summary: `Initial commit`
3. 点击 "Commit to main"
4. 点击顶部的 "Publish repository"
5. 配置：
   - Name: `feishu-avatar-webhook`
   - Description: `飞书群头像自动更新`
   - ✅ 取消勾选 "Keep this code private"（使用公开仓库）
6. 点击 "Publish repository"

### 第5步：完成

✅ 上传完成！你的仓库地址：
```
https://github.com/你的用户名/feishu-avatar-webhook
```

## 方案2：网页直接上传（无需安装任何软件）

### 第1步：创建GitHub仓库

1. 访问 https://github.com
2. 登录后点击右上角 "+" → "New repository"
3. 填写：
   - Repository name: `feishu-avatar-webhook`
   - Description: `飞书群头像自动更新`
   - 选择 **Public**
   - 不勾选任何选项
4. 点击 "Create repository"

### 第2步：上传文件

1. 在新仓库页面，点击 "uploading an existing file"
2. 打开文件夹 `E:\AI\修改头像`
3. 选择以下文件（按住Ctrl多选）：
   ```
   webhook_server_hybrid.py
   feishu_api.py
   config.json
   avatar.jpg
   requirements.txt
   render.yaml
   Dockerfile
   .gitignore
   README_GITHUB.md
   ```
4. 拖拽到浏览器窗口
5. 等待上传完成（可能需要1-2分钟）
6. 在底部填写：
   - Commit message: `Initial commit`
7. 点击 "Commit changes"

### 第3步：重命名README

1. 点击 `README_GITHUB.md` 文件
2. 点击右上角铅笔图标（编辑）
3. 将文件名改为 `README.md`
4. 点击 "Commit changes"

### 第4步：完成

✅ 你的仓库地址：
```
https://github.com/你的用户名/feishu-avatar-webhook
```
